|                      Le stringhe di diagnostica sono:                        |
--------------------------------------------------------------------------------
| > Quando un produttore soddisfa una porzione della sua richiesta             |
|   The Producer x has already written y requests                              |
|                                                                              |
|   x = identificatore del produttore                                          |
|   y = numero complessivo di richieste soddisfatte                            |
--------------------------------------------------------------------------------
| > Quando un produttore completa l'intera richiesta                           |
|   The Producer x completed writing phase                                     |
|                                                                              |
|   x = identificatore del produttore                                          |
--------------------------------------------------------------------------------
| > Quando un consumatore soddisfa una porzione della sua richiesta            |
|   The Consumer x has already read y requests                                 |
|                                                                              |
|   x = identificatore del consumatore                                         |
|   y = numero complessivo di richieste soddisfatte                            |
--------------------------------------------------------------------------------
| > Quando un consumatore completa l'intera richiesta                          |
|   The Consumer x completed reading phase                                     |
|                                                                              |
|   x = identificatore del consumatore                                         |
--------------------------------------------------------------------------------
